#  Password Strength Checker (Advanced)

A powerful Password Strength Checker built using Python.<br>
This tool not only checks password strength but also provides feedback, entropy calculation, crack time estimation, and strong password suggestions.<br><br>

This project is designed to demonstrate both basic and advanced password security concepts.


## Features

✔ Password strength classification (Weak / Medium / Strong)<br>
✔ Detailed feedback on improving passwords<br>
✔ Strong password generator<br>
✔ Password entropy calculation<br>
✔ Estimated time to crack password<br>
✔ Interactive CLI interface


## Concepts Used

• Regular Expressions (Regex)<br>
• Password Entropy Calculation<br>
• Brute Force Attack Estimation<br>
• Python String Handling<br>
• Conditional Logic


## Tech Stack

Python 3<br>
Built-in libraries only (re, math, random, string)
