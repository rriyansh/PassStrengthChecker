# Password Strength Checker

A simple Password Strength Checker built using Python.<br>
This program checks how strong a password is based on different security rules like length, uppercase letters, lowercase letters, numbers, and special characters.<br><br>

This is a beginner-friendly project made for practicing Python and basic security concepts.

## Features

✔ Checks password length<br>
✔ Detects uppercase letters<br>
✔ Detects lowercase letters<br>
✔ Detects numbers<br>
✔ Detects special characters<br>
✔ Gives password strength result (Weak / Medium / Strong)

## Tech Stack

Python 3
Regular Expressions (`re` module)
