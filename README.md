Hey! 👋
This is a simple Password Strength Checker that I built using Python.
The idea behind this project was to understand how password validation works and how we can check whether a password is weak, medium, or strong based on different conditions.
It checks things like:
Password length
Uppercase letters
Lowercase letters
Numbers
Special characters
How It Works <.>
The program takes a password as input and then checks:
If the password is at least 8 characters long
If it contains uppercase and lowercase letters
If it includes numbers
If it includes special characters
Based on these checks, it gives feedback like:
🔴 Weak
🟡 Medium
🟢 Strong

Built With
Python 3
re module (Regular Expressions)
